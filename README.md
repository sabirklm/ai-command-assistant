# AI Command Assistant
Convert natural language into **safe, executable CLI commands** using a local Ollama model.
Example:
```bash
njs run "commit my code with commit message setup completed"
```
Generates:
```bash
git commit -m "setup completed"
```
Then executes it after user confirmation.

* Commands are generated by AI
* Only allow-listed tools are executed
* User confirmation is required before execution
This prevents destructive or unsafe commands.
